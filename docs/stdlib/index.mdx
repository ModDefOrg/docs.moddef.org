# Standard library

Reusable ModDef packages that documents import by URI, versioned and resolved
against your package roots:

- **[Measurand catalog](/stdlib/measurands)** (`moddef:stdlib:measurands:1.0.0`):
  the canonical base quantities and units.
- **[OCPP aliases](/stdlib/ocpp-aliases)** (`moddef:stdlib:ocpp-aliases:1.0.0`):
  maps OCPP 1.6 measurand names onto ModDef tuples.
- **[SunSpec](/stdlib/sunspec)** (`moddef:stdlib:sunspec:1.0.0`): a starter
  SunSpec model mapping with discovery and scale factors.
- **[Core](/stdlib/core)** (`moddef:stdlib:core:1.0.0`): shared reusable enum types.

The stdlib lives in
[`moddef/stdlib`](https://github.com/ModDefOrg/moddef/tree/main/stdlib).
